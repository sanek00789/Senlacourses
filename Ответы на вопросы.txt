1. 	База данных - это совокупность данных, которые сведенных в одну систему и имееют определенную структуру.

2.  Реляционной БД - это тип базы данных, которая структурированна с помощью ключей для каждой записи, которые помогаю быстро находить связи и ориентироваться в этой базе данных.

3. 	Объекты - это таблицы базы данных, поля -это столбцы, записи - это строки.

4. 	Между полями могут быть связи master-detail и lookup.

5. 	Мастер-детейл это тип связи между полями объектов, при которой одно поле является основным, а второе дочерним. 
Данная связь накладывает жесткие ограничения, при которых основное поле контролирует свойства дочернего.
	Связи lookup – это тип связи между полями, при котором одно поле является источником информации для второго поля.

6. roll-up summary - это тип поля который объединяет информацию из зависимых полей.
